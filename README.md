# kkn-kkpDesktop
Aplikasi Pendaftaran KKP &amp; KKN adalah aplikasi berbasis Java yang mempermudah administrasi KKP dan KKN. Dengan fitur login multi-level, manajemen data master, pendaftaran, evaluasi, dan laporan, aplikasi ini mendukung admin, mahasiswa, dan dosen. Menggunakan arsitektur MVC dan database MySQL
